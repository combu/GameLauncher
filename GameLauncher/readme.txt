GameLauncher説明書

概要

ゲームの一覧を表示して起動する何か

仕様

表示されるゲーム名は設定ファイルの1行目で指定したディレクトリにあるサブディレクトリの名前が使用される。
空白にするとランチャーのあるディレクトリが指定される。

起動されるEXEファイルは各ゲームのディレクトリ内を探索して最初に見つかったファイルになる。
ランチャーが最小化されるのは、ランチャーによって起動されたプロセスが終了するまでの間なので
そのプロセスがまた別のプロセスを起動する場合には起動したプロセスが終了するまで待機している必要がある。

表示される説明は各ゲームのディレクトリ内にある説明書に準ずる名前のテキストファイルが表示される。
以下の正規表現で検索される。
(read\\s?me|(取扱?い?)?説明?書?|(始|初|はじ)め?に|(遊|あそ)び(方|かた))\\.txt

設定ファイル名はsetting.txtで、1行目にゲームの入っているディレクトリを指定し、
2行目はテーマ(黒ならblack白ならwhite)を指定し、3行目は「終了」メニューを表示するかどうかをtrueかfalseで指定する。
設定は壊れているとデフォルトの設定で上書きされる。

※readme.txtとsetting.txtのエンコードはShift-JISで保存すること

例1: C:\Gamesにゲームが入っているとして、白テーマで起動したい場合

setting.txt
----------
C:\Games
white
true
----------

例2: ランチャーと同じディレクトリにゲームが入っていて、黒テーマで起動したい場合

setting.txt
----------

black
true
----------

例3: ランチャーと同じディレクトリにゲームが入っていて、終了メニューを表示しないで起動したい場合

setting.txt
----------

black
false
----------

以下先輩からのメッセージ
・CDを焼くときには必ず最初に1枚試しに焼いて大丈夫か確認すること
・要らないファイル(Thumbs.dbとかdesktop.iniとかDXライブラリのログファイルとか)はなるべく消した方が良い
・ゲームがすべてRelease版でコンパイルされているか確認すること
・上記2つの作業はFileCheckerが全自動でやってくれるので活用して欲しい
